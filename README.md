# Employee Performance Analysis

## Overview

This project is developed using Python to analyze employee performance data. It demonstrates the complete data analysis workflow, including data loading, data cleaning, statistical analysis, and data visualization. The project helps identify top-performing employees, compare departmental performance, and understand the relationship between salary, experience, and performance.

## Project Workflow

### Step 1: Data Loading and Exploration

* Creates an employee dataset using Pandas.
* Displays the first few records of the dataset.
* Checks dataset shape.
* Identifies missing values.
* Generates statistical summaries.

### Step 2: Data Cleaning

* Replaces missing values using the median.
* Removes duplicate records.
* Converts Employee ID to string format for consistency.

### Step 3: Data Analysis

* Calculates the average performance score for each department.
* Identifies the top-performing employees.
* Computes the correlation between:

  * Tenure and Performance Score
  * Salary and Performance Score

### Step 4: Data Visualization

The project generates five professional charts:

* Performance Score Distribution (Histogram)
* Department-wise Average Performance (Bar Chart)
* Salary vs Performance (Scatter Plot)
* Performance vs Tenure (Line Chart)
* Correlation Matrix (Heatmap)

### Step 5: HR Executive Report

The final analysis provides insights that can help HR teams evaluate employee performance, compare departments, and support data-driven decision-making.
